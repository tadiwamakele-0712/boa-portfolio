# B.O.A Portfolio

Portfolio site for **Brothers of Africa (B.O.A)** — web development, digital platforms, and community projects by Tadiwa Makele.

## Live site

https://tadiwamakele-0712.github.io/boa-portfolio/

## Open locally

Open `index.html` in your browser, or run a local server from this folder:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`

## Tech

HTML5, CSS3, JavaScript
